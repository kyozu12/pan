About Me

Hello. My name is Mohammad Irfan Bin Mohammad Zawawi. I'm a student Diploma in Information Technology in Kolej Poly-Tech Mara (KPTM) at Batu Pahat, Johor. This site is created as part as my assignment in subject Web Programming to demostrate currency conversion betweeen Won and Ringgit. This assignment introduces students to the basics of HTML, CSS and Java Script to design a mini-WEBSITE.

The specific focus of the assignment is on currency conversion. This implies that the website will have functionality allowing users to input values, perform calculations, and view the converted results between the South Korean Won and Malaysian Ringgit.

In summary, the assignment not only serves as a practical introduction to HTML, CSS, and JavaScript but also emphasizes the practical application of these skills in designing a mini-website with a specific focus on currency conversion. The project aims to equip students with foundational skills that are crucial in the field of web programming and development.
